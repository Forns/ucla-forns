Your Week 5 Classwork

In class, you (as a group) designed programs to match some example specifications. Those answers are listed herein.

Example One: stringArrayPrint

  // Design a function stringArrayPrint that does 
  // nothing but print the comma and space-separated 
  // values of the first n values of an array of strings.
  void stringArrayPrint (string a[], int n) {
      cout << "{";
      for (int i = 0; i < n; i++) {
          cout << "\"" << a[i] << "\"";
          if (i < n - 1) {
              cout << ", ";
          }
      }
      cout << "}" << endl;
  }
