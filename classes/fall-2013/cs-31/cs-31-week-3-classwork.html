Your Week 3 Classwork

In class, you (as a group) designed programs to match some example specifications. Those answers are listed herein.

Example One: The Big N

Design a program that writes an 11 by 11 capital letter N made up of the character 'N'.

  #include <iostream>
  #include <string>
  using namespace std;
  
  /*
   * Design a program that writes 
   * an 11 by 11 capital letter N 
   * made up of the character 'N'.
   */
  
  // Example: 5x5 case
  // N   N
  // NN  N
  // N N N
  // N  NN
  // N   N
  
  int main () {
      // Assume SIZE must be odd
      const int SIZE = 11;
      
      // For loop that goes through
      // SIZE rows...
      for (int i = 0; i < SIZE; i++) {
        
          // And loop that goes through
          // SIZE columns
          for (int j = 0; j < SIZE; j++) {
              if (j == 0 || j == SIZE - 1) {
                  cout << "N";
              } else if (j == i && j != 0 && j != SIZE - 1) {
                  cout << "N";
              } else {
                  cout << " ";
              }
          }
          
          cout << endl;
      }
  }

Example Two: The Friendly Computer

Design a program that continues to ask the user for their name. Every time they give you the empty string, ask them more forcibly to provide an answer. If they don't answer 3 times, terminate with a an angry goodbye.

  #include <iostream>
  #include <string>
  using namespace std;
  
  /*
   * Design a program that continues to ask 
   * the user for their name. Every time they 
   * give you the empty string, ask them more 
   * forcibly to provide an answer. If they 
   * don't answer 3 times, terminate with a 
   * an angry goodbye.
   */
  int main () {
      // Want a counter to check which level
      // of angst your progam has achieved
      int counter = 0;
      string notAngry = "What is your name? ",
             angerLevel1 = "Tell me your name... ",
             angerLevel2 = "Spit it out. ",
             prompt,
             name;
      
      do {
          // Prompt for them to enter the name
          switch (counter) {
              case 0:
                  prompt = notAngry;
                  break;
              case 1:
                  prompt = angerLevel1;
                  break;
              case 2:
                  prompt = angerLevel2;
                  break;
          }
          cout << prompt;
          getline(cin, name);
          if (name == "") {
              counter++;
          } else {
              break;
          }
      } while (counter < 3);
      
      if (counter == 3) {
          cout << "Fine, have it that way." << endl;
      }
  }

Example Three: Capsizer

Get it? Because it turns caps and lowercase upside-down?

Write a program that queries the user for a string, swaps all lowercase letters for uppercase ones, and then prints out the result.

  #include <iostream>
  #include <string>
  #include <cctype>
  using namespace std;
  
  /*
   * Write a program that queries the user for a string,
   * swaps all lowercase letters for uppercase ones,
   * and then prints out the result.
   */
  int main () {
      string s;
      char c;
      
      // Ask for a string
      cout << "Gimme dat string ";
      getline(cin, s);
      cout << endl;
      
      // Size of string == s.length() == s.size()
      for (int i = 0; i < s.size(); i++) {
          // If it's uppercase, convert to lower
          if (isupper(s[i])) {
              c = tolower(s[i]);
              cout << c;
          } else {
              c = toupper(s[i]);
              cout << c;
          }
      }
      cout << endl;
  }
