Introduction to Arrays

We make lists of things every day, for example:

  1. Shopping lists are lists of storebought items.

  2. The even integers between 0 and 100 (inclusive) is a list of 51 integers.

  3. The list of things you can make a list out of is a list of list items... woah...

These lists have a variety of properties.

  1. These lists have a number of separate items that are usually of the same type.

  2. Each item has a relative position in the list (e.g. a first item, second item, and last item on a to-do list).

  3. The list has a length, designating the number of items that are currently on it.

  4. If I'm writing my list on paper, I only have so much room to write, so our lists might have a maximum size.
